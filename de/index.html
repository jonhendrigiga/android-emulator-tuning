Optimierungsreihenfolge

  1. VT aktivieren (+50-100%, sonst 1 Kern)
  2. CPU auf halbe physische Kerne
  3. RAM <=50% physisch
  4. fps auf 30-60 fixieren
  5. OpenGL/DirectX wechseln
  6. SSD + Hintergrund/AV schließen
  7. dedizierte GPU, Win-Update KB4100347 entfernen

Anti-Sperre (ehrlich)

  • pro Instanz eigene IP (SOCKS5-Proxy)
  • IMEI/Android-ID/MAC zufällig
  • zufällige Tippverzögerung, gestaffelte Logins
  • Sync-Controller in Ranked aus (wird erkannt); nur Farm

FAQ

Speicherleck im Leerlauf?

RAM pro Instanz begrenzen, gesamt <=80%, Neustart per ADB-Skript; nicht 24h durchlaufen.

VT an, aber VT belegt?

Hyper-V/Windows-Sandbox/Kernisolierung (Speicherintegrität) deaktivieren, dann neu starten.

VT lässt sich nicht aktivieren / BSOD?

Im BIOS Intel VT/AMD-V(SVM) an, Hyper-V/WSL2/VMware-Konflikte aus, Secure Boot aus.

LDPlayer oder MuMu?

Multi/Farm/schwacher PC -> LDPlayer; einzeln hohe Qualität/Anime/Mac -> MuMu.
