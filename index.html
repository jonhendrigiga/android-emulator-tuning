卡顿优化固定顺序

  1. 开 VT（提 50–100%，不开只能 1 核）
  2. CPU 设物理核一半
  3. 内存 ≤ 物理 50%
  4. 帧率按需锁 30–60
  5. 切 OpenGL/DirectX 渲染
  6. 装 SSD + 关后台/杀软

常见问题

多开内存占满/泄漏怎么办？

每实例设内存上限(挂机 2-3GB)、总量≤物理 80%、ADB 脚本定时重启；别 24h 持续挂机。

VT 开了还卡/「VT 被占用」？

关 Hyper-V / Windows 沙盒 / 内核隔离「内存完整性」(它会霸占 VT)，再重启。
